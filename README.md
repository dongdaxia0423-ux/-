# 🕒 工業計時儀 Pro (V3.4 修正版)

這是一個專為 Windows 設計的高精度計時工具，採用沉浸式工業儀表風格介面。

## 🌟 核心特色
* **智能校準邏輯**：防止輸入超標數字（例如輸入 66:66:66 將自動修正為 23:59:59）。
* **雙模式切換**：支援「鬧鐘模式」與「倒數計時器」，並可同時監控多個任務。
* **高精度碼表**：支援毫秒級記錄，並提供單圈（Lap）趨勢分析（進步/退步提醒）。
* **台北時區同步**：自動偵測並顯示台北標準時間 (CST)。

## 🚀 如何使用
1. **下載 EXE**：前往 `dist` 資料夾下載 `工業計時儀_Pro.exe`。
2. **直接執行**：無需安裝 Python，雙擊即可啟動。
3. **設定任務**：在面板輸入時間後按下 `EXECUTE` 即可開始監控。
